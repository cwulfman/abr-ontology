#+title: ABR Information Vocabulary
This document accompanies the formal ontology.

The Vocabulary captures two layers of information:
  - Information explicitly /captured/ on a form in form fields;
  - Information explicitly /required/ to request an absentee ballot, as specified on the form.

* Per-State Restrictions
The ontology a class for each state form that we have.  These classes
are defined as subclasses of oset:ABRForm with owl:Restrictions defining what type of information each form captures.
